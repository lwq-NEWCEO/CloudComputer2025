# [Number of Digit One][title]

## Description

Given an integer `n`, count _the total number of digit_`1` _appearing in all
non-negative integers less than or equal to_ `n`.



**Example 1:**
            Input: n = 13    Output: 6    

**Example 2:**
            Input: n = 0    Output: 0    



**Constraints:**

  * `0 <= n <= 109`


**Tags:** Math, Dynamic Programming, Recursion

**Difficulty:** Hard
