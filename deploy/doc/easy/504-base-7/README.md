# [Base 7][title]

## Description

Given an integer `num`, return _a string of its**base 7** representation_.



**Example 1:**
            Input: num = 100    Output: "202"    

**Example 2:**
            Input: num = -7    Output: "-10"    



**Constraints:**

  * `-107 <= num <= 107`


**Tags:** Math, String

**Difficulty:** Easy
